Animations spil

Den store opgave i Tema 03 var at vi ved brug af HTML, CSS og JavaScript skulle kode vores eget spil. Det var første gang, at vi blev introduceret for JavaScript og vi skulle ved brug af dette tilføje animation til vores elementer. Vi lærte blandt andet, at kunne placere elementer med CSS og give dem transformation, så de kunne placere sig de rigtige steder i forhold til spilet. Spillet indeholdte elementer som "tænd" og "sluk" for lyden, tidsbegrænsninger, point og liv, hvilket også var funktioner som vi fik til at fungere ved brug af JavaScript. Vi brugte blandt andet Toggle funktionen, if els funktionen, this, random og timer.
